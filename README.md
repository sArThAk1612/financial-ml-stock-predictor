
# Financial ML Stock Predictor

This project predicts next-day stock price movement using:

- Historical stock data
- News sentiment analysis
- Machine learning models

The pipeline generates a report containing:

• Predicted price  
• Expected percentage move  
• Confidence score  
• Sentiment summary  
• Top news headline

## Tech Stack

Python  
Pandas  
Scikit-learn  
NLP Sentiment Analysis
